# Zubaz Next.js App Router

This project is a Next.js App Router app (using the `src/app` directory).

## Requirements

- Node.js (LTS recommended)

## Development

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm start
```

## Project structure

- `src/app/**`: routes (App Router)
- `src/components/**`: UI/sections
- `public/assets/**`: static assets (CSS, images, fonts)
